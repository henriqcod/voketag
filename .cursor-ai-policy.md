CRITICAL ROUTING RULES
--------------------------------------------------------------------

1. Blockchain logic MUST be generated by GPT-5 only.
2. Authentication & Authorization logic MUST be reviewed by Claude.
3. Any file touching encryption MUST be validated by GPT-5.
4. Public API exposure MUST trigger security review.
5. Bulk refactoring MUST be done by Claude.
6. High-volume code generation MUST use DeepSeek.

If task involves:
- Scale > 1M requests/day → GPT-5 required
- Financial or compliance context → GPT-5 required
- Government or pharma data → mandatory security review

--------------------------------------------------------------------
SECURITY PRIORITY PROTOCOL
--------------------------------------------------------------------

If any ambiguity exists regarding:
- Data integrity
- Encryption
- Authentication
- Blockchain anchoring

Cursor MUST default to GPT-5.

Security over speed.
Architecture over convenience.
Auditability over shortcuts.

2. No secrets in frontend
